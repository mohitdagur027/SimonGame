Simon Game

Instructions

1. Press any key or Click on Start button to start the game.

2. The game will start by lighting up one of the four colored buttons (red, blue, green, yellow) and playing a corresponding sound.

3. Repeat the signal by pressing the same button.

4. Simon will add another button to the sequence and flash the new button.

5. Repeat these signals by pressing the buttons in order.

6. Continue playing as long as you can repeat each sequence of signals correctly.

7. If you fail to repeat a sequence exactly, the game will end.
